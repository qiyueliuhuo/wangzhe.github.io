应用数理统计

发表于 2021-01-04 更新于 2021-01-05 阅读次数： Valine：

学习数理统计总结。宗旨：用最少的语言去理解数理统计对于问题分析的应用和方法，尽量总结对于某些问题场景一般性思路。

第一章：数理统计的基本概念

概念思维导图

常用公式及相关结论

统计量

  • 样本均值：\(\bar{X}=\frac{1}{n} \sum_{i=1}^{n} X_{i}\)
  • 样本方差：\(S^{2}=\frac{1}{n} \sum_{i=1}^{n}\left(X_{i}-\bar{X}\right)^{2}=\frac{1}{n} \sum_{i=1}^{n} X_{i}^{2}-\bar{X}^{2}\)
  • 修正样本方差：\(S^{* 2}=\frac{1}{n-1} \sum_{i=1}^{n}\left(X_{i}-\bar{X}\right)^{2}\)
  • 样本标准差：\(S=\sqrt{S^{2}}\)
  • 样本\(k\)阶原点矩：\(A_{k}=\frac{1}{n} \sum_{i=1}^{n} X_{i}^{k}\)
  • 样本\(k\)阶中心矩：\(B_{k}=\frac{1}{n} \sum_{i=1}^{n}\left(X_{i}-\bar{X}\right)^{k}\)
  • 休息一会...
  • 样本中位数：\(M e=\left\{\begin{array}{ll}X_{((n+1) / 2)}, & n \text { 为奇数 } \\ \frac{1}{2}\left(X_{(n / 2)}+X_{(n / 2+1)}\right), & n \text { 为偶数 }\end{array}\right.\)
  • 样本极差：\(R=X_{(n)}-X_{(1)}\)

抽样分布

“统计量的概率分布，所有抽样的样本组成的统计量服从的分布”

  • 若随机变量 \(X_{1}, X_{2}, \cdots, X_{n}\) 相互独立,且都服从标准正态分布, 则随机变量 \(X=\sum_{i=1}^{n} X_{i}^{2}\)服从自由度为 \(n\) 的 \(\chi^{2}\) 分布 \(,\) 记为 \(X \sim \chi^{2}(n)\). 点击查看卡方分布图

第二章：参数估计

概念思维导图

常用公式及相关结论

矩估计

  • 用一阶：\(\bar{X} \stackrel{\text { 令 }}=E X\)。 解释：即用样本的均值等于总体\(X\)的期望（均值）。
  • 用二阶：\(\frac{1}{n} \sum_{i=1}^{n} X_{i}^{2} \stackrel{\text { 令 }}{=} E\left(X^{2}\right)\)

注意：

  • 当估计的参数只有一个时，一阶不能用，用二阶。

  • 计算\(E\left(X^{2}\right)\)时，要用到\(E\left(X^{2}\right) = D(X) + [E(X)]^{2}\)，这个公式根据\(D(X)=E\left(X^{2}\right)-[E(X)]^{2}\)得到，记后者更加容易些。解释：方差\(X\)平方的均值减\(X\)均值的平方。

极大似然估计

基于这样的原理：“参数为多少时，观测值出现的概率P最大”
1. 写似然函数 \(L\left(x_{1}, x_{2}, \cdots, x_{n} ; \theta\right)=\left\{\begin{array}{l}\prod_{i=1}^{n} p\left(x_{i} ; \theta\right), \\ \prod_{j=1}^{n} f\left(x_{i} ; \theta\right) .\end{array}\right.\)
2. 求参数 \(\left\{\begin{array}{l}\text { 有驻点 }: \frac{\mathrm{d} L}{\mathrm{~d} \theta}=0 \text { 或 } \frac{\mathrm{d} \ln L}{\mathrm{~d} \theta} \stackrel{\text { 令 }}{=} 0 \Rightarrow \hat{\theta} \\ \text { 无驻点(单调) : 用定义. }\end{array}\right.\)

区间估计

相关例题

  1. 设工件长度\(X \sim N\left(\mu, \sigma^{2}\right)\),今抽取10件测量其长度,得到如下数据(单位:cm) \[ 24.2,23.8,24.0,25.5,25.6,24.8,23.6,24.5,25.3,24.7 \] 在置信度\(0.95\)下,求参数\(\mu, \sigma^{2}\)的置信区间。

解题思路：

  1. 由数理统计相关知识，均值和方差未知，故\(\mu\)的置信度为\(1-\alpha\)的置信区间为 \[ \left(\bar{X}-\frac{S^{*}}{\sqrt{n}} t_{a / 2}(n-1), \bar{X}+\frac{S^{*}}{\sqrt{n}} t_{a / 2}(n-1)\right) \]
  2. 根据样本值计算\(\bar{X}、S^{*}\)，查表得到\(t_{0.025}(9)\)，代入上式子即可以求出置信区间。

总结：

所有区间估计一般的套路：
1. 先分类，比如属于正态分布，那么书上给了几种估计正态分布参数的类型，根据问题所属的种类，直接应用结论。 2. 计算结论中所需计算的一些值，如\(S^{*}\)，这些值是根据样本计算得来。
